# Code for my MagTag
It's a weather display, clock, and name tag! Wow! Written in CircuitPython.

<img width="4608" height="3456" alt="magtag" src="https://github.com/user-attachments/assets/65e4a532-d786-406d-9098-e3c58f409d16" />

## Features:
It's pretty basic but uh:
 - Button 1 toggles frontlight
 - Button 2 brings you to the weather display!
 - Button 3 is the clock
 - Button 4 is the name tag function

## Motivation
Idk I had bought a MagTag a while ago and never used it, so I decided to finally use it for something useful.
Turns out using it as a clock isn't as great as I'd imagined, but otherwise I'm pretty happy with it.

# Installation
[Install CircuitPython](https://learn.adafruit.com/adafruit-magtag/circuitpython) onto your MagTag, then copy the code.py file from this repository onto the MagTag. Bam that's it. Easy peasy.
